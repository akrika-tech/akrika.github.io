🚀 Internship Program

🎓 Internship Domains Offered

  • 🧠 AI / Machine Learning: Python, model training, projects, deployment
  • 🌐 Web Development: HTML, CSS, JavaScript, GitHub, hosting
  • 💻 Software Development: Backend logic, real project flow, Git
  • 📱 App Development: Mobile UI/UX and apps using Flutter / Android Studio
  • 🖥️ C / C++ Programming: Logic building, algorithms, coding fundamentals
  • 🐍 Python: Scripting, automation, data handling, mini projects
  • ☕ Java: OOP, console apps, reusable Java modules

🔁 Multi-Level Internship Model

  • 🔹 Level 1 – Unpaid Internship: Open to all applicants. Free project work + certificate upon completion.
  • 🔹 Level 2 – Shortlisting Round: An online test to evaluate knowledge and interest.
  • 🔹 Level 3 – Interview or Final Project: Top candidates are shortlisted for premium internships (may include interview or advanced evaluation).

👑 Top Performers may be selected for internships worth up to ₹1,00,000 in value — absolutely free. Your performance, dedication, and test results will determine your level.

📜 Certificates will be issued by Akrika Technologies, a Govt. Registered MSME (Udyam-HR-19-0059373).
